- デプロイ: Cloudflare Pages
- メンテナンスフリー（紋様の形は不変）
- **差別化**: 和柄に特化したパターンジェネレーターは海外にほぼ存在しない

## パターンエディター（`/:pattern`）

### カスタマイズオプション
- **色1（前景）**: カラーピッカー + HEX入力
- **色2（背景）**: カラーピッカー + HEX入力
- **色3（アクセント）**: 一部の紋様のみ（任意）
- **スケール**: パターンの繰り返しサイズ（スライダー: 20px〜200px）
- **線の太さ**: SVGのstroke-width（1〜5px）
- **回転**: 0° / 45° / 90° / 135°
- **不透明度**: 0〜100%

### プレビュー
- 大きなプレビューエリア（400x400px以上）にパターンをタイル状に表示
- SVGの `<pattern>` + `<rect>` でリアルタイムレンダリング
- 背景色付きプレビュー（パターンが実際のデザインでどう見えるかイメージしやすい）

### ダウンロード
- **SVG**: そのまま（ベクター、無限拡大可能）
- **PNG**: Canvas APIで描画してダウンロード（サイズ選択: 512/1024/2048px）
- **CSS**: `background-image: url("data:image/svg+xml,...")` のインラインSVGコード
- **Figma / Sketch用**: SVGファイル（そのままインポート可能）

### コピー機能
- SVGコード
- CSSコード（background-imageのインラインSVG）
- color-convへのリンク（「Edit colors in Color Converter」）

## パターンのSVG実装

各紋様は純粋な関数として実装する:
```typescript
// src/patterns/seigaiha.ts
interface PatternParams {
  color1: string     // 前景色
  color2: string     // 背景色
  color3?: string    // アクセント色（任意）
  scale: number      // パターンサイズ
  strokeWidth: number
  rotation: number
  opacity: number
}

function generateSeigaiha(params: PatternParams): string {
  // SVGの<pattern>要素の中身を返す
  return `<circle cx="..." cy="..." r="..." .../>...`
}
```

### 紋様の数学的定義
- 市松: 正方形の交互配置
- 麻の葉: 正三角形の組み合わせ（60°の線が6本交差）
- 青海波: 同心円弧の重なり
- 鱗: 正三角形のタイル
- 七宝: 円の1/4弧の組み合わせ
- 亀甲: 正六角形のタイル
- 矢絣: 平行四辺形の交互配置
- etc.

すべてSVGの基本図形（circle, path, line, rect, polygon）で描画可能。
